# Condo API

Server API for condo resource management.

## Requirement
Ruby 2.7.1

Postgres 12.4

## Setup / installation

1. Clone repo
2. `bundle install`
3. `rails db:setup`
4. `rails db:migrate`
5. Copy `.env.sample` to `.env` and enter information
5. `rails server -e development`

## Linting

Run `rubocop` to lint.

## Testing

Run `rails test` to test.
